# 🌿 Paradise Nursery

A React + Redux shopping application for houseplants. This project was built as the final assignment for the Frontend Development course.

## Features

- 🌱 Browse houseplants by category (Indoor, Outdoor, Succulents)
- 🛒 Add plants to shopping cart with quantity management
- 📱 Responsive design with beautiful landing page
- 🔄 Redux state management for cart operations

## Technologies

- React 18
- Redux Toolkit
- React Router DOM
- Vite
- CSS3

## Getting Started

```bash
npm install
npm run dev
```
